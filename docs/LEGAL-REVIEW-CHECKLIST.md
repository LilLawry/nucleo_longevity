# Legal Review Checklist

> Everything a qualified lawyer must sign off before commercial launch. Each item
> is tagged so it can be tracked. Nothing here is legal advice.

## YMYL / medical claims
- [ ] **LEGAL REVIEW REQUIRED** — confirm no entry reads as medical advice; the
      A–F grade is framed as answering a question, not a recommendation.
- [ ] Prescription/injectable entries (`metformina`, `rapamicina`,
      `lithium-low-dose`, prescription `retinoids`) carry supervision language and
      no purchase CTA. (**REGULATORY REVIEW REQUIRED** for regulated substances.)
- [ ] Disclaimers on `/disclaimer` and every molecule page are sufficient.

## Advertising / affiliate
- [ ] Affiliate disclosure (`/funding-and-affiliations`, `/prezzi`, `/confronto`)
      meets local advertising-disclosure rules.
- [ ] "Sponsored profile" labelling meets rules; sponsored ≠ verified is explicit.
- [ ] **AGREEMENT REQUIRED** before any real affiliate/merchant feed is enabled.

## Connect / directory
- [ ] No real company/person is listed without authorisation; demo data labelled.
- [ ] No official partnership asserted without a signed agreement
      (**AGREEMENT REQUIRED**).
- [ ] Verification labels (`unverified → verifiedPartner`) are backed by a
      documented process before any "verified" is shown on real data.
- [ ] Logos only with `logoUsageSource` permission recorded.

## Privacy / data
- [ ] Public `/privacy` reviewed against PRIVACY.md; processor list + transfers
      confirmed. **LEGAL REVIEW REQUIRED.**
- [ ] Newsletter double opt-in + consent records compliant.
- [ ] No health data collected anywhere (verified in forms).

## IP / content
- [ ] No copied PubMed abstracts; citations link out, summaries are original.
- [ ] Third-party content (images, data) properly licensed.

## Trademark / brand
- [ ] "Nucleo Longevity" / "Núkleo" usage cleared; NUKLEO name not used.

## Sign-off
- [ ] Counsel review complete and dated.
- [ ] Regulatory review complete for any regulated-substance content.
