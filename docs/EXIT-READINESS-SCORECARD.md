  newsletter subscribers with source; returning users.

- **Data moat:** 12–15 pillars `verified` with real PMIDs; outcome relations;
  original comparisons; freshness cadence.

  documented workflows.

  trademark search + (if chosen) filing; privacy DPA/records.
- **Technology:** add tests + Zod schemas; a11y pass; performance budget.
- **Strategic value:** proven B2B relationships/leads; a defensible vertical.
